# Registro de Usuarios Seguro

## Descripción
Proyecto web que implementa un formulario de registro de usuarios aplicando buenas prácticas de seguridad, validaciones en frontend y backend, y almacenamiento seguro de contraseñas.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Funcionalidades
- Registro de usuarios
- Validaciones del lado del cliente y servidor
- Hash seguro de contraseñas
- Prevención de inyección SQL
- Manejo seguro de errores

## Instalación
1. Clonar el repositorio
2. Colocar el proyecto en la carpeta htdocs
3. Iniciar Apache y MySQL
4. Importar la base de datos
5. Acceder desde http://localhost/
